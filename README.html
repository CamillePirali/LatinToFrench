LatinToFrench - guide d'utilisation

Prérequis

Avant toute chose, assurez-vous que :

- Perl est installé sur votre ordinateur ;
- Les fichiers LatinToFrench.pl et simple_lex.tsv (au minimum) ont été téléchargés ;
- Ces fichiers ont été enregistrés dans le même dossier ;
- Les modules String::Similarity et IO::Prompt::Simple sont installés sur votre ordinateur. Si ce n'est pas le cas, tapez les commandes suivantes dans votre terminal : "cpanm String::Similarity" et "cpanm IO::Prompt::Simple".

Exécution du programme

Dans votre terminal, naviguez jusqu'à l'endroit où se trouvent les documents mentionnés ci-dessus. Si vous utilisez Windows, tapez la commande "chcp 65001" et appuyez sur Enter.
Entrez ensuite votre commande selon le mode que vous souhaitez utiliser :

1) View
Syntaxe : perl LatinToFrench.pl view gaudi*am
Retrace les différentes étapes de l'évolution du mot latin passé en argument et imprime les meilleurs matchs trouvés dans la base de données Lexique3.

Exemple de sortie:

### Prétraitement ###
Découpe en syllabes : gau-di*-am (3 syllabe(s))
Place de l'accent tonique : g'audi*am

### Latin vulgaire ###
Evolution au 1er siècle : g'audja
Evolution au 5ème siècle : g'aujja

### Gallo-roman ###
Evolution au 8ème siècle : (dZ)'oi3

### Français archaïque ###
Evolution au 12ème siècle : (dZ)'ue3

### Ancien français ###
Evolution au 13ème siècle : Zw'a3

### Moyen français ###
Evolution au 16ème siècle : Zw'a

### Français moderne ###
Evolution au 18ème siècle : Zw'a

### Meilleurs matchs ###
joie,joies

2) Evol
Syntaxe : perl LatinToFrench.pl evol gaudi*am joie
Le premier argument est un mot latin, le second l'évolution en français moderne que vous avez trouvée (tous les deux dans leur forme orthographique). Le programme imprime un message selon que votre réponse correspond au résultat qu'il a obtenu, en est très proche ou ne correspond pas du tout.
Vous aurez ensuite la possibilité de voir l'évolution du mot étape par étape et d'effectuer des modifications si vous le souhaitez.

Exemple de sortie:

Félicitations ! Votre réponse est correcte

Souhaitez-vous voir l'évolution du mot ?
(n/o) : n

3) Test
Syntaxe : perl LatinToFrench.pl test test_set.txt
Ce mode permet d'évaluer la performance du programme sur un corpus de test. Le fichier doit comprendre deux colonnes séparées par des tabulations, la première contenant les mots latins à faire évoluer et la seconde leur évolution correcte.

Exemple de sortie:

Test de performance en cours...

Performance: 89.6551724137931 %


Remarques importantes

- Les mots passés en arguments doivent être dans leur forme orthographique : le programme s'occupera lui-même de les convertir en forme phonétique.
- Les voyelles brèves doivent être suivies par "*" pour permettre le bon fonctionnement du programme. Si vous n'êtes pas certain.e de la longueur des voyelles, vous pouvez essayer différentes combinaisons.
(Indice : les voyelles en hiatus sont généralement brèves.)
